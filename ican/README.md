# Execution

## TwoLevel
```
./binfile |data_file.wl| |stopword_file.sw| < |queries.txt|
```

## ICAN
```
./binfile |data_file.wl| |stopword_file.sw| < |queries.txt|
```

## META
```
./binfile |data_file.txt| |queries.txt| |distancia de edição|
```

## IncNGTrie
```
./binfile -t |distancia de edição| -d |data_file.txt| < |queries.txt|
```
